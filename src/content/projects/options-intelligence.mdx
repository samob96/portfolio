## Overview

A personal options trading dashboard that pulls live data from Schwab's API, ranks contracts using a rating model, and surfaces suggestions across multiple timeframes.

## What it does

- Pulls live quotes + chain data from Schwab
- Ranks contracts using a custom scoring model with auto-calibration
- Streams contract updates through a live feed
- Archives sessions for later analysis

## Why I built it

Personal interest in algorithmic trading plus a genuine itch to work with a real-time financial data stream end-to-end.
